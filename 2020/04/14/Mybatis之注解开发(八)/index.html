注解开发保存和更新功能

 @Insert("insert into user(username,address,sex,birthday)values(#{username},#{address},#{sex},#{birthday})")    
 void saveUser(User user); 
 @Update("update user set username=#{username},address=#{address},sex=#{sex},birthday=#{birthday} where id=#{id}")
 void updateUser(User user);

 @Delete("delete from user where id=#{id}")    
 void deleteUser(Integer userId);
@Select("select * from user where username like #{username}")    
List<User> findUserByName(String name);  
//模糊查找

当实体类属性名和数据库列名不一致时

我们可以加入以下注解完成实体类属性和数据库列名的映射

 @Select("select * from user")
    @Results(value = {
            @Result(id = true, column = "id", property = "userId"),
            @Result(column = "username", property = "userName"),
            @Result(column = "address", property = "userAddress"),
            @Result(column = "sex", property = "userSex"),
            @Result(column = "birthday", property = "userBirthday"),
    })
    List<User> findAll();

针对多个查询方法，再分别给它们加上注解的话代码重复性就太高了

因此，我们可以给注解加上唯一标识，让其他方法去引用它

@ResultMap(value = {"userMap"})是标准写法，因为可以引用多个

如果只引用一个的话可以用以下简略写法

（原理：value本身就可以省略；只有一个元素的话{}也可以省略）

@ResultMap("userMap")

注解开发多表一对一查询

查询账户时同时查询对应的用户

 @Select("select * from account")
@Results(id = "accountMap", value = {
        @Result(id = true, column = "id", property = "id"),
        @Result(column = "uid", property = "uid"),
        @Result(column = "money", property = "money"),
        @Result(property = "user", column = "uid", one = @One(select = "com.zzq.dao.IUserDao.findById", fetchType = FetchType.EAGER))
})
List<Account> findAll();

其中@Result注解的one字段有select和fetchType两个属性（详情可以看源码）
select属性：配置根据主键查询的方法名的路径
fetchType属性：配置延迟加载/立即加载

注解开发一对多的查询配置

我们要在User实体类中添加一对多关系映射

@Select("select * from user")
@Results(id = "userMap", value = {
        @Result(id = true, column = "id", property = "userId"),
        @Result(column = "username", property = "userName"),
        @Result(column = "address", property = "userAddress"),
        @Result(column = "sex", property = "userSex"),
        @Result(column = "birthday", property = "userBirthday"),
        @Result(property = "accounts", column = "id", many = @Many(select = "com.zzq.dao.IAccountDao.findAccountByUid", fetchType = FetchType.LAZY))
})
List<User> findAll();
